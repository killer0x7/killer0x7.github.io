Hexo搭建博客

发表于 2019-11-22 Valine：

物理机环境：windows10 安装git node.js

GitBash下进行如下操作

npm install -g cnpm --registry=https://registry.npm.taobao.org //cnmp比nmp速度快

cnpm install -g hexo //全局安装hexo
mkdir blog //站点主目录
cd blog
hexo init//初始化hexo
git clone https://github.com/theme-next/hexo-theme-next //安装next主题
config.yml theme:next scheme: Gemini //改变主题和风格
hexo new "new article" //新文章

cnpm install --save hexo-deployer-git //安装部署插件
更改站点的配置文件中的deploy type:git repo:仓库地址 branch:master

hexo clean //清除缓存
hexo s //开启本地服务器
